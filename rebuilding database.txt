to rebuild a database with sqlalchemy:

- drop database, then create it
- delete all migration versions
- start the shell
- run: flask db migrate
- run: flask db upgrade
- run: flask seed all

heroku commands:

heroku run -a splitzyapp flask db migrate
heroku run -a splitzyapp flask db upgrade
heroku run -a splitzyapp flask seed all

to only drop the data from the databse:
- run: flask undo all
